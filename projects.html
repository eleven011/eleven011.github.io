my projects

AnimalHelper

This is a voice-chatbot I started in Voice Technologies through PSU. The objective is to help pet owners seek advice on various situations concerning their pets.

AnimalHelper is built using DialogFlow, so I used the platform and wrote my code in JavaScript.

InstaServe

InstaServe is a mobile application I developed with my Capstone team. This was a 6 month project with 7 other team members using React-Native.

The purpose of this app is to increase Covid-friendliness for restaurants and their patrons by allowing the patron to handle all interactions with wait staff from their phone.

Frupal

Frupal is a web-based game I made with a group of classmates for CS300 at PSU. This app is built almost entirely in JavaScript. Click on the image if you're interested in trying it out!

Ping CLI

My_ping is a command line interface ping application. I followed a tutorial and made this as part of an internship application. This app sends a "ping" to a host in order to test how reachable that host is. My_ping is written entirely in C.

Carduino

Carduino is an arduino-driven car project created by me and my fellow mentors for our mentees. My coworkers and I worked for an organization at PSU that supports traditionally underrepresented students in STEM called "Scholars for Success in STEM" or S^3. This was our end of year project.
